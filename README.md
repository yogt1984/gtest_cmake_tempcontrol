# gtest_cmake_tempcontrol

A sample project illustrating how to perform unit testing with GoogleTest and CMake

## Requirements 
CMake Version 2.8 +

## Tested
on Ubuntu 20.04.6 LTS

## Building

~~~
mkdir build
cd build
cmake ..
make
~~~

## Run test cases

from project folder

~~~
./build/test/testtemperaturecontrol/testtemperaturecontrol
~~~



## Run app 

from project folder

~~~
./build/main/myApp
~~~
